Esta actividad consiste en manejar una gran base de datos de albergues. De los albergues se guarda su provincia, fecha de apertura, nombre, municipio... Todos estos datos se leen de los archivos albergues.csv y capacidad_albergues.csv mediante la clase AlberguesFileDAO, que es la que lee los datos y los registra correctamente. Se implementa Comparable en Albergue para que se puedan ordenar de forma natural en colecciones, y GestionAlbergue implementa dos métodos principales, mostrarAlbergues() y mostrarPlazasProvincia(), que se podrán usar tras registrar los datos.
