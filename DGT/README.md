Este es un examen que hicimos principalmente de manejo de ficheros y excepciones. Consiste en leer de diferentes archivos lo que varios alumnos habían respondido en distintos tests de examen del carnet de conducir. También teníamos un archivo con las soluciones, y, después de procesar los datos, había que escribir los puntajes en otro archivo. Creamos un DAO externo para acceder a los ficheros de datos (DireccionTraficoDAO), que hereda de una interfaz (IDireccionTraficoDAO), y que hace uso de la excepción DAOException. También tenemos una clase Candidato que usamos para guardar a los diferentes candidatos y sus puntajes. Por último creamos otra clase externa CalculadoraPuntajes, y todo se maneja desde la clase principal DireccionTraficoApp.
